ARQUITECTURA DE LA APLICACIÓN DE INVERSIÓN CRYPTO

1. VISIÓN GENERAL
La aplicación es una plataforma de inversión en USDT que permite a los usuarios conectar sus wallets de MetaMask, ver sus saldos y realizar inversiones en diferentes planes con distintos porcentajes de retorno.

2. COMPONENTES PRINCIPALES

a) Gestión de Wallet (WalletContext)
- Maneja la conexión con MetaMask
- Gestiona el estado de la conexión y la cuenta del usuario
- Controla los saldos de USDT
- Implementa las operaciones de inversión
- Mantiene un registro de las inversiones activas

b) Gestión de Tema (ThemeContext)
- Permite cambiar entre modo claro y oscuro
- Persiste la preferencia del usuario
- Se integra con el sistema operativo para el tema inicial

c) Interfaz de Usuario
- Navbar: Barra de navegación con conexión de wallet y cambio de tema
- Hero: Sección principal con información y botón de conexión
- InvestmentPlans: Muestra los planes de inversión disponibles
- Dashboard: Panel de control para ver inversiones activas
- Footer: Pie de página con enlaces y información adicional

3. FLUJO DE OPERACIÓN

a) Conexión de Wallet:
1. Usuario hace clic en "Connect Wallet"
2. Se verifica la instalación de MetaMask
3. Se solicita acceso a la cuenta
4. Se obtiene y muestra el saldo de USDT
5. Se cargan las inversiones activas

b) Proceso de Inversión:
1. Usuario selecciona un plan
2. Se abre modal de inversión
3. Usuario ingresa cantidad a invertir
4. Se valida el saldo y monto mínimo
5. Se procesa la inversión
6. Se actualiza el dashboard

4. CARACTERÍSTICAS DE SEGURIDAD

- Validación de conexión de MetaMask
- Comprobación de saldos suficientes
- Manejo de errores en transacciones
- Políticas de reconexión automática
- Actualización en tiempo real de cambios de cuenta

5. PLANES DE INVERSIÓN

a) Conservador:
- Retorno: 5% APY
- Duración: 30 días
- Inversión mínima: 100 USDT

b) Balanceado:
- Retorno: 10% APY
- Duración: 60 días
- Inversión mínima: 500 USDT

c) Crecimiento:
- Retorno: 15% APY
- Duración: 90 días
- Inversión mínima: 1000 USDT

6. TECNOLOGÍAS UTILIZADAS

- React con TypeScript
- Ethers.js para interacción con blockchain
- Tailwind CSS para estilos
- Context API para gestión de estado
- MetaMask como proveedor de wallet

7. CONSIDERACIONES TÉCNICAS

- La aplicación está preparada para múltiples redes de Ethereum
- Manejo de cambios de red y cuenta
- Sistema de temas adaptativo
- Interfaz responsive
- Validaciones en tiempo real
- Manejo de errores robusto

8. LIMITACIONES ACTUALES

- Solo soporta USDT como moneda de inversión
- Requiere MetaMask instalado
- Los datos de inversiones son simulados
- No incluye histórico de transacciones
- No implementa contratos inteligentes reales
